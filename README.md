# chinese-2026wc-guide

本仓库用于归档和发布多个独立的 HTML 页面，内容围绕 2026 年世界杯相关的信息整理与指南。仓库本身不针对任何特定域名或网站，仅作为页面资源的静态存储与分发用途。

## 项目简介

该项目旨在收集、整理并发布一系列与 2026 年世界杯相关的 HTML 页面，包括但不限于赛事信息、观赛指南、队伍介绍、赛程概览等。所有页面均为独立 HTML 文件，不依赖后端服务，可通过静态托管方式直接访问。

## 目录说明

仓库根目录下存放所有 HTML 页面文件，目录结构如下：

```
/
├── index.html          # 入口页面
├── page-about.html     # 关于页面
├── page-schedule.html  # 赛程页面
├── page-teams.html     # 队伍页面
├── page-guide.html     # 观赛指南页面
└── assets/
    ├── css/            # 样式文件
    ├── js/             # JavaScript 脚本
    └── images/         # 图片资源
```

各页面文件命名清晰，便于直接链接或引用。

## 页面归档说明

所有 HTML 页面均为独立归档，内容不涉及具体商业推广或诱导点击。页面内容以信息整理和呈现为主，数据来源均为公开可查信息。归档版本会随赛事进展进行更新，但不对更新频率做承诺。
